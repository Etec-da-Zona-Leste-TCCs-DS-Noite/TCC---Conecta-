# TCC---Conecta-

## Descrição

O projeto **TCC—Conecta-** tem como objetivo facilitar a comunicação entre alunos e a secretaria escolar, promovendo agilidade e transparência nos processos internos do ambiente acadêmico.

Este sistema está sendo desenvolvido pelos alunos do segundo módulo do curso de Desenvolvimento de Sistemas da Etec da Zona Leste, em caráter de Trabalho de Conclusão de Curso (TCC).

## Funcionalidades

- Comunicação direta entre estudantes e secretaria
- Solicitação e acompanhamento de documentos
- Notificações sobre pendências e atualizações
- Histórico de solicitações efetuadas
